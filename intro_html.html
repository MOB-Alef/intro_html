Introdução ao HTML

O HTML, sigla para HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a linguagem base da construção de páginas internet.

Criado nos primórdios da web, o HTML define a estrutura e o conteúdo de uma página, organizando textos, imagens, links, tabelas, formulários e outros elementos visuais que o usuário visualiza no navegador. Diferentemente de linguagens de programação como Python ou JavaScript, o HTML não possui lógica de programação — ele apenas marca e organiza o conteúdo. Cada elemento HTML é definido por tags (etiquetas), como para títulos principais, para parágrafos e para imagens. Essas tags indicam ao navegador como exibir cada parte do conteúdo.
